# PART-09: Пустые состояния, загрузка, ошибки

---

## 1. Загрузка (global)

- При проверке сессии на старте: полноэкранный или узкий блок с `auth.loading`.
- Внутри модулей: флаг `pending` — скелетоны или текст «Загрузка…» в рамках панели.

---

## 2. Пустые данные

| Место | Поведение |
|-------|-----------|
| Герой: нет квестов | Текст `hub.questEmpty` + список доступных |
| Герой: стрики 0 | Пилюли с «0 дн.» (не прятать блок) |
| Задачи: нет на сегодня | Сообщение из `todo.*` с призывом добавить |
| Финансы: нет операций | Пустой список с CTA добавить операцию |
| Тело: нет замеров | Подсказка пройти в замеры |

---

## 3. Ошибки

- Одна строка или баннер вверху/внизу секции красным/вторичным цветом.
- Повтор действия — там, где в коде есть retry (не везде).

---

## 4. Disabled / «Скоро»

- Колокол уведомлений на «Герой» — disabled.
- Блок «Скоро» на «Герой» — иконки disabled.
- Любые future features — визуально приглушены, без обязательного tooltip в v2.

---

## 5. Чеклист для макетов

- [ ] Для каждого ключевого экрана есть **frame «Empty»** и **frame «Error»**.
- [ ] Состояние загрузки не ломает высоту карточек (скелетоны равной высоты).
