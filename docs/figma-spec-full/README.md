# Полное ТЗ интерфейса Real Hero для Figma / дизайна / вёрстки

**Назначение:** единый источник правды о том, **что должно быть нарисовано и как ведёт себя UI** веб-приложения **Real Hero**, чтобы макеты **совпадали с реальным функционалом** (React SPA + REST API).

**Важно про «10 000 строк»:** осмысленное ТЗ такого размера в одном файле **не поддерживаемо** (шум, противоречия, невозможность ревью). Здесь сделано следующее:

1. **PART-01 … PART-10** — сжатая, но детальная спецификация по разделам.
2. **APPENDIX-*** — полные копии проектных `ТЗ_*.md` и **всех крупных файлов строк интерфейса** (`finance`, `body`, `hub`, `todo`, `common`) в виде `.ts.txt`; вместе с PART-файлами это даёт **тысячи строк** содержательного текста для Figma AI и дизайна.
3. Общее число строк (включая приложения с полным `index.css` и `FinanceModule.tsx`): **около 11 000+** — см. команду подсчёта:  
   `Get-ChildItem docs/figma-spec-full -Recurse -File | Get-Content | Measure-Object -Line` (PowerShell).

См. также `APPENDIX-README.md` в этом каталоге.
