# PART-02: Оболочка приложения и навигация

Источник в коде: `src/components/AppShell.tsx`, `src/context/ShellTabContext.tsx`, `src/lib/shellTabs.ts`, `src/components/AppNavIcons.tsx`.

---

## 1. Маршрутизация верхнего уровня

| URL | Экран | Защита |
|-----|--------|--------|
| `/` | Лендинг | Нет |
| `/login` | Вход | Нет |
| `/register` | Регистрация | Нет |
| `/app/*` | `AppShell` (все 5 разделов) | Да (`Protected`: cookie/сессия) |

Всё приложение после входа — **одна оболочка** без смены URL между разделами (карousel внутри `/app`).

---

## 2. Карусель разделов (shell)

- Контейнер: `.shell__carousel` — горизонтальный скролл с **привязкой к ширине экрана**: один «слайд» = 100% ширины трека.
- Каждый слайд: `section.shell__slide` (модификаторы `--finance`, `--body`, `--todo` для своих фонов при необходимости).
- **Индекс активного слайда** синхронизирован с нижней навигацией и с `ShellTabContext` (`activeIndex`, `goToTab`).

---

## 4. Верхняя панель (header)

### 4.1. Режим «не вкладка Герой» (`active !== 0`)

- **Слева:** email пользователя (усечение ellipsis, `title` полный email).
- **Справа:** переключатель языка **RU | EN** (две кнопки, активная подсвечена), кнопка **«Выйти»**.

### 4.2. Режим «вкладка Герой» (`active === 0`)

Класс шапки: `shell__bar--hub-mockup` + обёртка `shell__bar-hub-wrap`.

**Первая строка (`shell__bar-hub-row`):**

- **Слева:** заголовок уровня `h1`, текст из i18n `shell.tabHub` (**«Герой»** / Hero).
- **Справа:** две квадратные кнопки-иконки:
  - **Шестерёнка** — переход на вкладку индекс **4** (настройки). Иконка: `ShellNavIcon id="settings"`.
  - **Колокол** — пока **disabled**, подсказка «уведомления скоро» (`hub.bellAria`).

**Вторая строка (`shell__bar-hub-meta`):**

- Слева направо: усечённый **email**; блок **RU / EN**; **«Выйти»** (компактные стили).

Макет должен показывать **оба ряда** на узкой ширине без наложения текста.

## 5. Нижняя навигация

- Контейнер: `nav.shell__nav`, `aria-label` = `shell.navMain`.
- Кнопки: `shell__nav-btn`, активная — модификатор `--active`.
- Внутри: иконка `shell__nav-ic` + текст `shell__nav-tx`.

| Порядок | `navIcon` (AppNavIcons) | Ключ подписи `shell.*` |
|---------|-------------------------|-------------------------|
| 1 | `hub` | `tabHub` |
| 2 | `body` | `tabBody` |
| 3 | `todo` (ракета в макете) | `tabDevelopment` |
| 4 | `finance` | `tabFinance` |
| 5 | `settings` | `tabSettings` |

---

## 6. Поведение при смене языка

- Мгновенная смена локали `ru` | `en` через контекст i18n.
- Все подписи нижней навигации и шапки **перерисовываются**; длина строк на EN может быть длиннее — предусмотреть перенос или уменьшение кегля в макете.
