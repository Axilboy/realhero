# PART-01: Дизайн-система (токены, типографика, сетка)

Документ описывает **визуальный язык v2** для Real Hero. Реализация в коде: `src/index.css` (`:root`), компонентные классы.

---

## 1. Философия

- **Одна тёмная тема** в продакшене v2; светлая тема не описана.
- **Мобильный первичный контекст:** основная колонка ограничена по ширине, контент читается без горизонтального скролла (кроме намеренных каруселей).
- **Данные важнее украшений:** плотность умеренная; числа и денежные суммы читаемы; интерактивные элементы ≥44×44px там, где ожидается тап.
- **Акцент синий** для ссылок, активных вкладок, первичных действий в спокойных экранах; **зелёный/красный** — семантика доход/расход и позитив/ошибка, не как «второй бренд».

---

## 2. Цветовые токены (CSS variables)

| Токен | Hex | Назначение |
|-------|-----|------------|
| `--rh-bg` | `#0f1115` | Фон всего приложения за колонкой |
| `--rh-surface` | `#14171c` | Карточки, панели |
| `--rh-surface-elevated` | `#1a1d24` | Приподнятые блоки, инпуты |
| `--rh-border` | `#2a2f38` | Основные границы |
| `--rh-border-muted` | `#252a33` | Разделители, мягкие границы |
| `--rh-accent` | `#8ab4ff` | Ссылки, активная вкладка, CTA-акцент |
| `--rh-text` | `#e8eaed` | Основной текст |
| `--rh-text-secondary` | `#c4c7cc` | Вторичный текст |
| `--rh-text-muted` | `#9aa0a6` | Подписи, подсказки |
| `--rh-nav-bg` | `#12151a` | Фон нижней навигации и части шапок |
| `--rh-income` | `#81c995` | Доход, позитивные суммы |

## 3. Радиусы

| Токен | Значение | Применение |
|-------|----------|------------|
| `--rh-radius-card` | `12px` | Карточки, крупные контейнеры |
| `--rh-radius-btn` | `10px` | Кнопки, чипы крупные |
| Пилюли / чипы | `999px` | Стрики, фильтры |

---

## 4. Типографика

- **Семейство:** `system-ui, -apple-system, "Segoe UI", Roboto, sans-serif` (в макете: Inter или SF Pro как приближение).
- **Заголовок экрана:** крупный, жирный, `color: --rh-text`, часто `letter-spacing` слегка отрицательный для крупных кеглей.
- **Подзаголовки секций:** uppercase + `letter-spacing` + приглушённый цвет (пример: `.hero__card-title`, `.finance` секции).
- **Числа:** для денег и таблиц — **tabular lining figures** где доступно (`font-variant-numeric: tabular-nums`).
- **Минимальный кегль для длинных списков:** не ниже **11–12px** эквивалента при нормальном контрасте; предпочтительно 13–14px для вторичного текста.

---

## 5. Сетка и отступы

- **Горизонтальные отступы колонки:** типично `1rem` (16px) у краёв shell; в коде встречается `padding` у `.shell`, `.screen`.
- **Вертикальный ритм:** кратность **4px** или **8px** (отступы между секциями 12–16px, между крупными блоками 16–24px).
- **Максимальная ширина:** `428px` — фрейм «логический iPhone 13 Pro Max»; на десктопе колонка **по центру** viewport, фон `--rh-bg` по краям.

---

## 6. Эlevation и тени

- v2 в основном **плоский**; глубина за счёт **границы** и **лёгкого градиента** на карточках героя.
- Модальные окна: затемнение фона (`rgba(0,0,0,0.5)` или аналог) + поверхность `--rh-surface-elevated`.

---

## 7. Иконки (стиль)

- **Нижняя навигация приложения:** монохромный **stroke**, ~22px, `currentColor`.
- **Поднавигации модулей:** уменьшенный размер (~20px), тот же принцип.
- **Эмодзи** допустимы в тексте кнопок и пилюль на экране «Герой» (конкретные — в PART-03).

---

## 8. Состояния интерактивных элементов

| Состояние | Визуал |
|-----------|--------|
| Default | Как в токенах |
| Hover (desktop) | Слегка светлее фон кнопки или сильнее border |
| Active / pressed | Чуть темнее фон |
| Focus-visible | Кольцо `outline: 2px solid var(--rh-accent)` + `offset` |
| Disabled | `opacity` ~0.5–0.65, `cursor: not-allowed`, без hover-эффекта |
| Loading | Кнопка с `disabled` + спиннер или текст «…» |

---

## 9. Z-index (логика слоёв)

От меньшего к большему (ориентир для Figma):

1. Фон приложения  
2. Контент скролла  
3. Нижняя навигация (фиксирована визуально внизу колонки)  
4. Шапка shell  
5. Модальные оверлеи  
6. Тосты / критические диалоги (если появятся)

Точные числа в коде не всегда заданы; в макете **не перекрывать** нижнюю навигацию контентом кроме намеренных sheet/modal.

---

## 10. Motion

- Переключение вкладок: **гладкий скролл** карусели (уже в продукте).
- Прогресс-бары: **короткая** анимация ширины (~0.25s ease).
- Избегать длинных анимаций при открытии списков; приоритет — отзывчивость.

---

## 11. Чеклист для Figma

- [ ] Все цвета из таблицы токенов заведены как **styles**.
- [ ] Радиусы 10 / 12 как **variables**.
- [ ] Текстовые стили: Title / Body / Caption / Overline (uppercase).
- [ ] Отдельный **frame 428px** с названием `App / Mobile / Column`.
- [ ] Компонент **Button**: Primary / Secondary / Ghost / Danger (если используется).
- [ ] Компонент **Card**: default, dashed (квесты), tinted (мини-карточки героя).
