# Эталонные мокапы v2 (PNG)

Сюда кладутся **согласованные скриншоты** интерфейса Real Hero для сверки с вёрсткой.

## Именование

Рекомендуемый шаблон: `v2-rh-NN-<экран>.png`, например:

- `v2-rh-01-hero.png` — экран «Герой»
- далее по модулям — по договорённости с `ТЗ_UI_МОКАПЫ_V2.md`

## Как сверять

- Ширина колонки приложения в коде: **428px** (`--rh-shell-max`).
- Сравнивать в браузере с тем же масштабом или в DevTools с эмуляцией устройства той же ширины.
- Токены цветов: `src/index.css` (`:root`, `--rh-*`).

## Пустая папка

Если PNG ещё не добавлены в репозиторий, положите файлы локально и закоммитьте — тогда их смогут использовать и люди, и инструменты в среде разработки.

См. также: `ТЗ_ФАЗА_ВИЗУАЛА.md`, `ТЗ_UI_МОКАПЫ_V2.md`.
