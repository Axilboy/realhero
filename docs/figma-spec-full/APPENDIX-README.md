# Приложения (полные копии исходников)

Файлы в этом каталоге с префиксом **`APPENDIX-*`** — это **не дублирование ради объёма**, а **точные копии** документов и файлов переводов из репозитория на дату сборки набора ТЗ. Они нужны Figma AI и дизайнерам, чтобы видеть **все строки интерфейса** и **юридически зафиксированные ТЗ** без открытия IDE.
