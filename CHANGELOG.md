# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).  
**Единый номер версии** веб-клиента: поле `version` в `package.json` (его же подхватывает UI и `src/version.ts`).

## [0.6.0] — 2026-03-28

### Добавлено

- **Дашборд из API** (`GET /api/v1/dashboard`): уровень, EXP, стрик, подсказки с учётом `settings.dashboardHints`, уведомления (монеты, стрик, квесты).
- **Квесты**: модели `UserStats`, `Quest` в Prisma; API `GET/POST /api/v1/quests`, `POST .../complete`, `DELETE .../id`; начисление EXP по формуле уровня, монеты, стрик по UTC-дням.
- Экран **`/quests`**: создание, выполнение, удаление, архив выполненных.
- Без сессии: дашборд предлагает войти; mock API удалён.

## [0.5.0] — 2026-03-28

### Добавлено

- Экран **`/profile`**: имя для обращения, почта (только чтение), настройка «подсказки на главном» (`settings.dashboardHints` в БД).
- API: **`PATCH /api/v1/me`** (обновление `displayName` и слияние `settings` JSON).
- Поле **`User.settings`** в Prisma (SQLite dev).
- Локально: **`POST /api/v1/auth/dev-login`** при **`DEV_RELAXED_AUTH=1`** и не-production — кнопка на `/login` только в Vite dev.
- Дашборд: приветствие с **именем из профиля**, если пользователь вошёл.

## [0.4.0] — 2026-03-28

### Добавлено

- Страница **`/login`**: вход через Google, Яндекс, VK (редирект на API OAuth).
- **`SessionProvider`** / **`useSession`**: `GET /api/v1/me`, выход через `POST /api/v1/auth/logout` (cookie `rh_session`).
- Верхняя панель: ссылки «Войти» / имя и «Выйти».
- Vite **proxy** `/api` → `http://127.0.0.1:3000` для локальной разработки (один origin с callback OAuth).

## [0.3.0] — 2026-03-28

### Добавлено

- Каталог **`api/`** — HTTP API (Fastify + Prisma) для веба, Telegram и будущих Android/iOS; префикс `/api/v1`.
- Модель **`User`** (заготовка под Telegram `telegramId`).
- Документ **`docs/ARCHITECTURE.md`** и обновление паспорта проекта.

## [0.2.0] — 2026-03-28

### Добавлено

- Верхняя панель с названием приложения и версией из `package.json`.
- Страница «История изменений» (`/changelog`) и ссылка с главного экрана.
- Обратный свайп с экранов модулей на центр (противоположное направление открытию).
- Скрипты быстрой заливки: `scripts/quick-push.sh`, `scripts/server-pull-deploy.sh`.
- Документ `deploy/temporary-https.md` (временный HTTPS / Telegram).

### Изменено

- Убраны дубли подсказок со стрелками на дашборде при наличии нижней навигации; упрощён блок «На сегодня» в mock API.

### Документация

- README: раздел «Быстрая заливка».
- Паспорт проекта: актуальная версия веб-клиента и политика версий.
